docs/decisions/index.md must never be edited by hand — it is auto-generated by scripts/generate-knowledge-index.sh after any ADR is added or changed. The planning workflow's opening codebase scan reads index.md to identify which ADRs are relevant before pulling full files; a stale or manually-patched index will cause the planner to miss or misroute decisions.
